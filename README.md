AutoPartsQA — Data Quality Analysis Platform
AutoPartsQA is a Streamlit‑based data quality dashboard designed to analyze, validate, and visualize issues in automotive parts datasets.
It performs automated checks for:

Missing values

Duplicate records

Invalid prices

Spelling inconsistencies

Brand/vehicle gaps

Overall data quality scoring

This project demonstrates practical skills in Python, Pandas, data validation, visualization, and dashboard development — ideal for data engineering and analytics roles.
